# RankUp — App

Application Expo (React Native + TypeScript).

Voir le [README principal](../README.md) pour la documentation complète.

## Commandes rapides

```bash
npm install          # Installer les dépendances
npx expo start       # Lancer en mode dev (mobile)
npx expo start --web # Lancer en mode web
npx jest             # Lancer les tests
```
